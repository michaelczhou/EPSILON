template<class State, class Value = double, class Deriv = State, class Time = Value, class Algebra = range_algebra, class Operations = default_operations, class Resizer = initially_resizer>
class boost::numeric::odeint::modified_midpoint< State, Value, Deriv, Time, Algebra, Operations, Resizer >

Implementation of the modified midpoint method with a configurable number of intermediate steps. This class is used by the Bulirsch-Stoer algorithm and is not meant for direct usage.
