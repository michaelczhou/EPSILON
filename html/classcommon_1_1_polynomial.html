template<int N_DEG>
class common::Polynomial< N_DEG >

Polynomial class, coeffs are stored in reversed order The parameterization is given by f(s) = coeff_(n) + coeff_(n-1)/1!^s + ... + coeff_(0)/(n!)*s^n f(s) = coeff_normal_order_(0) + coeff_normal_order_(1)^s + ... + coeff_normal_order_(n)*s^n coeffs are scaled to have straightforward physical meaning, also improve the efficiency when evaluating the derivatives.
